# Serwer Websockets



## Potrzebne oprogramowanie

- Maven
- Java 11+



## Konfiguracja i uruchomienie

-Aby uruchomic aplikację należy przejść do jej głównego katalogu i wpisać w terminalu `mvnw spring-boot:run`
